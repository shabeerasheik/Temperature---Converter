# 🌡️ Temperature Converter

This project was developed as part of my SkillCraft Technology Internship.

## 📌 Features
- Celsius → Fahrenheit
- Fahrenheit → Celsius
- Celsius → Kelvin
- Kelvin → Celsius
- Fahrenheit → Kelvin
- Kelvin → Fahrenheit

## 🛠️ Technologies Used
- Java
- HTML
- CSS
- JavaScript

## 🚀 How to Run
1. Compile and run the Java program.
2. Open `index.html` in your web browser.
